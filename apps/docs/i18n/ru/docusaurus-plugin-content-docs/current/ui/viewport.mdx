# Видимая область

Технология Telegram Web Apps позволяет отображать приложение разработчика
на разных платформах, что приводит к увеличению списка возможных устройств
пользователей. Учитывая этот факт, важно понимать, что одно устройство может
сильно отличаться от другого, и когда на ПК оно всегда будет занимать всю
видимую область, которая не изменяется, то на мобильных устройствах она
может быть динамическое.

Использование информации о видимой области приложения может помочь улучшить
качество приложения разработчика, а также повлиять на опыт пользователя.

Видимая область обладает следующими свойствами:

- Стабильность
- Развернутость
- Высота и ширина

## Мобильные устройства

В случае, если приложение разработчика открывается в мобильной версии Telegram
(Android и iOS), оно будет отображено в таком нативном компоненте как
`BottomSheet`. Он представляет собой вертикально перетягиваемый блок, который
может иметь высоту вплоть до высоты экрана.

Для разворота приложения пользователю достаточно потянуть блок с приложением
вверх. Более того, программно это же может сделать и разработчик приложения.

По умолчанию изображение отображается в неразвернутом состоянии, тогда оно
имеет минимальную доступную высоту. Для разворота приложения необходимо
использовать выделенный для этого [метод](../apps-communication/methods.mdx#web_app_expand).

В процессе перетягивания блока с приложением, видимая область считается
нестабильной. Под стабильностью мы пониманием отсутствие возможности
возникновения ситуации, при которой видимая область изменится в следующий
момент времени. Для разработчика это означает, что, возможно, он не должен
делать каких-либо визуальных изменений.

## Другие устройства

Другие устройства открывают Web App уже развернутым в окне малого размера.
Вызов метода для разворота приложения не создаст никакого эффекта.

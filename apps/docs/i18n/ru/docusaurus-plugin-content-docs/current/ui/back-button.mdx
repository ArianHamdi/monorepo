# Back Button

Основной задачей, которую выполняет Back Button, является предоставление
кажущимся нативным, способа перехода назад в истории переходов. Тем не менее
Telegram не ограничивает разработчика в способах применения Back Button
и позволяет ему обрабатывать событие клика на компонент как потребуется в
приложении.

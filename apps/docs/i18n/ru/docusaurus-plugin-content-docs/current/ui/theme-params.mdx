# Параметры темы

Web Apps это веб-приложения, которые должны выглядеть нативно. Это подразумевает
использование не только компонентов, которые мимикрируют под нативные, но и
так же цвета, используемые в нативном приложении Telegram.

Приложению разработчика передается список цветов, которые на
данный момент используется приложением Telegram. В свою очередь, приложение
разработчика должно использовать этот список для того, чтобы выглядеть
согласованно с приложением Telegram.

## Цвета фона и заголовка

Приложение разработчика отображается в нативном компоненте, которое в
упрощенном смысле состоит из 2 частей: заголовка и тела. Самому разработчику
предоставляется возможность управления фоновым цветом каждого из них.

Для того чтобы изменить их фоновые цвета, необходимо использовать следующие
методы:

- [Метод для задания цвета фона](../apps-communication/methods.mdx#web_app_set_background_color)
- [Метод для задания цвета заголовка](../apps-communication/methods.mdx#web_app_set_header_color)

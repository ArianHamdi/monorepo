# Popup

Popup - это компонент, который располагается поверх приложения TWA. Его
классическим случаем применения является запрос подтверждения пользователя
на выполнение какого-либо действия.

Разработчику предоставляется возможность задать заголовок, сообщение, а также
список кнопок, которые будут отображаться внутри всплывающего окна.

Говоря о кнопках, важно отметить, что каждая из них также поддаётся
кастомизации. Среди полей, доступных для изменения, находятся следующие:

- Идентификатор
- Текст
- Тип. Определяет цветовую гамму кнопки, а также предоставляет возможность
указывать локализованный текст для типовых случаев

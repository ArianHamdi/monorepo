Параметры темы

Web Apps это веб-приложения, которые должны выглядеть нативно. Это подразумевает использование не только компонентов, которые мимикрируют под нативные, но и так же цвета, используемые в нативном приложении Telegram.

Приложению разработчика передается список цветов, которые на данный момент используется приложением Telegram. В свою очередь, приложение разработчика должно использовать этот список для того, чтобы выглядеть согласованно с приложением Telegram.

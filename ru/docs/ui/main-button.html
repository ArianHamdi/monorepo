Main Button

Main Button представляет собой компонент, который обычно используется в случае необходимости выполнения какого-либо завершающего действия.

Примером случая применения этого компонента является отправка данных. Например, приложение TWA может подразумевать выбор товаров из каталога и заполнение корзины пользователя. В момент, когда в корзине появляются товары, разработчик может отобразить Main Button, клик на который вызовет создание и обработку заказа.

Это достаточно большой и в то же время простой компонент, который предоставляет узкий функционал. У разработчика есть возможность контролировать следующие свойства Main Button:

  • Текст
  • Цвет текста
  • Фоновый цвет
  • Видимость
  • Видимость лоадера внутри кнопки
  • Активность. Позволяет предотвращать клик на кнопку делая её неактивной
подсказка

В случае, если нажатие на кнопку производит действие, выполнение которого требует некоторого времени, рекомендуется отображать лоадер внутри Main Button. Это позволит пользователю понять, что приложение не зависло и в данный момент выполняет операцию.

  • Связанный метод
  • Генерируемое событие при клике на кнопку
